# J.Books Online Bookstore Web Project

This is a responsive **ReactJS** website for selling books online. It has a clean, professional design and uses **Bootstrap** for styling. The website works seamlessly on both desktop and mobile devices, providing a smooth user experience.

## Pages

The website includes the following pages:

1. **Home** – Main landing page with featured books and a search bar.  
2. **Books** – Dynamic page displaying the available books for sale.  
3. **Cart** – Page to view, update, or remove items added to the shopping cart.  
4. **Favorite** – Page to view books marked as favorites by the user.  
5. **Contact/About** – Page with information about J.Books and a contact form.  

## Features

- Responsive design (desktop and mobile friendly)  
- Bold **Navbar** with J.Books branding  
- Add, update, and remove books from the cart  
- Mark books as **favorites** and view them in the Favorite page  
- Clean UI with Bootstrap styling  
- Modular React components for easy maintenance
